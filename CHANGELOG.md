## [1.0.2] – 2026-04-25

### Added

- Initial release of the official PHP client.
- Type-safe bindings for the Tippiti OpenAPI 3.1 specification.
- `Tippiti::configure()` factory returning a configured `Configuration` for use with any Api class under `Tippiti\Api\Generated\Api\*`.
- Guzzle / PSR-7 based, framework-agnostic.
